# v1.0.0 Release Notes

🚀 **FM26 Kit Creator v1.0.0 - Official Launch Release**  
Professional desktop app for creating Football Manager 2026 kits. 

**Features:**  
- Advanced Canvas Editor (Fabric.js)  
- Direct FM26 Export  
- Cloud Sync & Backup  
- Team Collaboration  
- Batch Export  
- User Authentication  
- Payment Integration  
- Analytics  
- 24/7 Support  

**Free tier includes:**  
- Basic Canvas Editor  
- 5 Projects  
- Local Export (PNG/JPG)  
- Community Support  
- Desktop & Web Access  

**Pro tier:** $9.99/month with unlimited projects, advanced editing, batch export, cloud sync.  

**System Requirements:**  
- Windows 10+, macOS 10.13+, or Ubuntu 18.04+, 4GB RAM, 500MB disk space.  

Get started by cloning the repository: `git clone https://github.com/tonym6350-star/fm26-kit-creator.git`, then `npm install` and `npm start`.  
Download installers coming soon.
